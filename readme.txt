This plugin is for France based 0 8xx number display. Basically, it will add the mandatory fancy look for any 0 8xx that you will add to any page, programatically, by shortcode or in raw html

== Description ==

No offence if I write in french but this plugin is no use outside France
 
Le plugin "Numéros Spéciaux" reprend la charte graphique imposée par SVAplus (<a href="http://www.svaplus.fr/">www.svaplus.fr</a>) et vous permet, en un clic, d'installer avec l'affichage réglementaire vos numéros spéciaux sur votre site.

En suivant les <a href="https://je-suis-nul-en-informatique.fr/numeros-speciaux/">manips recommandées sur le site</a>, une fois le plugin installé, vous pouvez utiliser du HTML brut ou un shortcode comme indiqué à la suite.

Basile Bernard

== Installation ==

Upload the "Numéros Spéciaux" plugin to your blog, Activate it.

Use the shortcode like 

[numerosspeciaux numero="0 891 690 177" type="payant"   perso="0,50€/min" size="dessous"  ]
or
[numerosspeciaux numero="0 891 690 177" type="gratuit"   size="normal"  ]
or
[numerosspeciaux numero="0 891 690 177" type="normal"  size="small"  ]
or
[numerosspeciaux numero="0 891 690 177" type="normal"  size="dessous" color="defonce"  ]
or any combination of the above

1, 2, 3: You're done!
